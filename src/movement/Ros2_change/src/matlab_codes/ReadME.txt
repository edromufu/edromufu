Todos os scripts/funções possuem uma letra antes de seu nome, essas letras significam:

d -> Done, códigos que precisam possuir sua versão em Python e já possuem.

n -> Not, códigos que não precisam possuir sua versão em Python.

t -> Todo, códigos que precisam possuir sua versão em Python e ainda precisam ser feitos.

**DURANTE A PASSAGEM DOS CÓDIGOS COM PREFIXO "t" PARA PYTHON, NÃO ESQUECER DE RETIRAR O "t"**
